# Angular 12+ and Node.js SPA with Async Delay Processing

This repository features a decoupled Single Page Application (SPA) built using Angular, TypeScript, and an Express backend. It showcases Role-Based Access Control (RBAC), simulated asynchronous network latency, and custom dashboards for General Users and Administrators.
